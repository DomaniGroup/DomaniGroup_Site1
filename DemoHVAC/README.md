# Northstar Comfort Co. — Static HVAC Demo

This is a self-contained GitHub Pages package for the Northstar Comfort Co. HVAC demo.

## Deploy at `domanigroup.ai/DemoHVAC`

1. Copy the contents of this folder into a `DemoHVAC` folder in the GitHub Pages repository.
2. Commit and push the folder to the branch and directory used by GitHub Pages.
3. Open `https://domanigroup.ai/DemoHVAC/`.

The package uses relative asset paths, so it works from a subdirectory. No build step or framework runtime is required.

## Included demo interactions

- Booking form with date, time window, issue description, and picture upload
- Six-question instant quote engine
- AI receptionist demo with emergency escalation responses
- Financing calculator and prequalification flow
- Comfort Club membership flow
- Customer portal and dedicated owner dashboard previews
- Emergency mode, technician tracking, review filters, and responsive navigation

The scheduling, payments, AI responses, CRM data, and dashboard metrics are front-end demo behavior. Connect them to production services before using the site with customers.
